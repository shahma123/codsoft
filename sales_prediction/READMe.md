Sales Prediction

1.Objective
The objective of this task is to predict product sales based on advertising expenditure across different channels such as TV, Radio, and Newspaper.
This task demonstrates data preprocessing, exploratory analysis, model training, performance evaluation, and prediction
This project predicts product sales using advertising features.

2.Dataset
 Source: from Kaggle
 Columns:
⦁	 TV — Budget spent on TV ads
⦁	 Radio — Budget spent on Radio ads
⦁	 Newspaper — Budget spent on Newspaper ads
⦁	 Sales — Total product sales

3.Models implemented
⦁	 Linear Regression 
⦁	 Random Forest Regressor 

4.How to run
1. Open `sales_prediction_notebook.ipynb` or run the provided Colab cells.
2. If using Colab, upload the CSV or mount Google Drive.
3. Run all cells. Outputs:
   - `sales_model.pkl` (saved model)
   - `sales_processed.csv` (processed data used for training)
